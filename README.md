# Zelo App Chat - ReactJs

## Cách chạy

npm install
npm start

Chạy trên cổng 3000

## `Home`

-   `/`: trang chủ.
-   `/account/login`: trang login.
-   `/account/registry/email`: đăng kí email.
-   `/account/registry/phone`: đăng kí phone.
-   `/account/forgot`: quên mật khẩu.

## `Chat`

-   `/chat`: trang chat chính.
-   `/chat/friends`: trang bạn bè.
